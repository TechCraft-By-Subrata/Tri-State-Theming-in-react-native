## Theming Mastery: Implementing Tri-State in React Native

### Why This Course?
Most web-based theming tutorials focus on a simple binary toggle: **Light or Dark**. In professional mobile development, that isn't enough. To provide a truly native experience, your app must respect system-level preferences while still offering manual control.

This means if a user chooses Light or Dark, the app follows that selection. However, there is a third essential option: **System**. When selected, the app dynamically follows the mobile device’s theme—switching to Light mode or Dark mode automatically based on the OS settings.

In this course, we go beyond "demo apps." We dive into the architecture of **Tri-State Theming** (Light, Dark, and System), the exact architecture I implemented in my own **Water Tracker** app.
